# Radeon RX 7900 XTX Software Requirement Specification

## Thermal Architecture
- Maximum junction temperature: 110C.
- Vapor chamber cooling solution.
- **Troubleshooting Overheating**: If junction hits 110C+ consistently, check mounting pressure and case airflow. If repaste is required, escalate to Hardware Replacements.

## Drivers & Displays
- DisplayPort 2.1 support.
- **Troubleshooting Flickering**: If the user experiences screen flickering, first step is to use AMD Cleanup Utility and clean install the latest WHQL driver. If issue persists with multiple cables, escalate to GPU Drivers team.
