# AMD Adrenalin Software SRS

## Overlay and Features
- Contains Radeon Anti-Lag, Boost, and Super Resolution (RSR).
- **Troubleshooting Software Crashes**: If the UI fails to open after a Windows Update, Windows may have replaced the driver. 
- **Self-Service Step**: Tell the user to download the latest AMD Auto-Detect tool to fix the driver mismatch. If this fails, escalate to GPU Drivers team.

## Game Profiles
- Custom tuning profiles can be saved per game.
- **Troubleshooting Stuttering**: Disable Radeon Chill and FRTC (Frame Rate Target Control) if micro-stuttering occurs in competitive games.
